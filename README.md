# Reactor Skills

Skills follow the [Agent Skills](https://agentskills.io/) format.

## About

* [`skills/helios-prompts`](./skills/helios-prompts) - Craft effective prompts for Helios video generation
* [`skills/sana-streaming-prompts`](./skills/sana-streaming-prompts) - Turn rough edit intent into production-grade SANA-Streaming video-editing instructions
* [`skills/lingbot-world`](./skills/lingbot-world) - Author interactive LingBot video worlds, from quick prose prompts to full JSON virtual worlds
* [`skills/lingbot-2-prompts`](./skills/lingbot-2-prompts) - Turn a scene idea into LingBot-World V2 session prompts (base prompt + action slots + seed image)
* [`skills/reactor-sdk`](./skills/reactor-sdk) - Build real-time video AI apps with the Reactor JS and Python SDKs
* More coming soon as we approach public beta

## Install

```bash
npx skills add reactor-team/reactor-skills
```
